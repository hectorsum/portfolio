# Portfolio UI Kit — README

## Overview
Interactive click-through prototype for Hector Herrera's portfolio website.
Four pages: Home, About, Projects, Contact.

## Files
- `index.html` — Main interactive prototype (loads all components)
- `Header.jsx` — Site navigation with active state
- `HeroSection.jsx` — Home page hero
- `ProjectCard.jsx` — Project card component
- `ProjectsGrid.jsx` — Projects page grid
- `AboutSection.jsx` — About page
- `ContactSection.jsx` — Contact page
- `Footer.jsx` — Site footer

## Design Specs
- Design width: 1280px (desktop-first, responsive)
- Fonts: DM Serif Display (display), Outfit (body), JetBrains Mono (code/tags)
- Background: #0D0D0D
- Accent: #C8965A

## Usage
Open `index.html` in a browser. Click nav links to switch pages.
All interactions are simulated — no real backend.
